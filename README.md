# Pulmonary Nodule Classification

This project classifies pulmonary nodules using:

- Fleischner Society 2017 Guidelines
- Lung-RADS v2022

## Features

- Extracts nodule size from radiology reports
- Determines patient risk category
- Applies guideline-based classification
- Generates management recommendations

## Technologies

Python
Pandas
Regex
Google Colab

## Output

Final CSV containing classification for 20 clinical cases.
